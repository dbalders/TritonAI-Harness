# Contributing

We are not taking outside PRs right now.

For bugs, install problems, docs fixes, feature requests, or questions, open a GitHub issue.

Useful issue details:

- What you expected
- What happened
- OS and app version, if relevant
- Screenshots or logs, if they help

Do not post API keys, tokens, private UCSD data, student or employee data, or other sensitive information in public issues.

If a report needs private details, keep the issue minimal and use the appropriate UCSD/TritonAI internal channel for the sensitive part.
